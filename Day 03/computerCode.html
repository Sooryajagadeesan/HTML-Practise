CODE -to specify any computer program code.
int number_1 = 10; int number_2 = 20; int sum = number_1 + number_2; printf("Sum is %d", sum);
Use pre to preserve the spaces in the Code.
int number_1 = 10;
int number_2 = 20;
int sum = number_1 + number_2;
printf("%d", sum);

VAR -to specify the variables especially while writing formulas.
Area of triangle : (1/2)*base*height
